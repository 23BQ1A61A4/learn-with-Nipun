HTML Introduction

HTML stands for HyperText Markup Language. It is the standard language used to create web pages. HTML describes the structure of a webpage using elements. It tells the browser how to display content such as headings, paragraphs, images, links, tables, and forms.

HTML is not a programming language. It is a markup language, which means it uses tags to define elements within a document. Every webpage you see on the internet is built using HTML along with CSS and JavaScript.

Example

<!DOCTYPE html>
<html>
<body>
<h1>Welcome</h1>
<p>This is my first webpage</p>
</body>
</html>

HTML Elements

An HTML element usually consists of a start tag, content, and an end tag. For example, a paragraph element starts with <p> and ends with </p>.

Some elements do not have closing tags. These are called empty elements, like <br> and <hr>. 

<p>This is a paragraph</p>
<br>
<hr>

HTML Attributes

Attributes provide additional information about HTML elements. They are always included in the opening tag.

For example, the href attribute in the anchor tag specifies the link destination. 

<a href="https://google.com">Visit Google</a>

HTML Headings

HTML provides six levels of headings from <h1> to <h6>. <h1> is the most important heading and <h6> is the least important. 

<h1>Main Heading</h1>
<h2>Sub Heading</h2>

HTML Paragraphs

The <p> tag defines a paragraph. Browsers automatically add space before and after paragraphs. 

<p>This is a paragraph.</p>

HTML Styles

The style attribute is used to add CSS styles directly to HTML elements. You can change color, font size, background color and more. 

<p style="color:red;">Red Text</p>

HTML Formatting

Formatting tags are used to make text bold, italic, highlighted, or important. 

<b>Bold</b>
<i>Italic</i>
<mark>Highlight</mark>

HTML Links

Links are created using the <a> tag. The href attribute specifies the URL. 

<a href="https://example.com">Click Here</a>

HTML Images

Images are inserted using the <img> tag. The src attribute specifies the image path. 

<img src="https://via.placeholder.com/150">

HTML Tables

Tables are used to display data in rows and columns using <table>, <tr>, <th>, and <td> tags. 

<table border="1">
<tr><th>Name</th><th>Age</th></tr>
<tr><td>John</td><td>20</td></tr>
</table>

HTML Lists

Lists can be ordered (<ol>) or unordered (<ul>). 

<ul>
<li>Apple</li>
<li>Banana</li>
</ul>

HTML Forms

Forms are used to collect user input. The <form> element wraps input fields. 

<form>
<input type="text">
</form>

HTML JavaScript

JavaScript can be added inside HTML using the <script> tag. It is used to make webpages interactive. 

<button onclick="alert('Hello')">Click</button>
