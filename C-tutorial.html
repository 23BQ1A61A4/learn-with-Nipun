C Programming - Full Course

C is one of the most powerful and widely used programming languages. It is used for system programming, embedded systems, OS development, and even game engines.

This tutorial is designed for beginners and will cover all important concepts with examples.

C Intro

C is a general-purpose programming language developed by Dennis Ritchie in 1972. It is known for its speed, efficiency, and low-level memory access.

Why learn C?

  • Base for C++, Java, Python concepts
  • Used in Operating Systems
  • Very fast execution
  • Best for understanding memory and pointers

C Get Started

To run C programs, you need a compiler like GCC, Turbo C, or online compilers.

First C Program

#include <stdio.h>

int main() {
    printf("Hello World!");
    return 0;
}

C Syntax

A C program consists of:

  • Header files (#include)
  • main() function
  • Statements ending with semicolon ;
#include <stdio.h>

int main() {
    // code here
    return 0;
}

C Output

C uses printf() function to print output. 

printf("Welcome to C");

C Comments

Comments are ignored by compiler. They are used to explain code. 

// Single line comment

/*
Multi line comment
*/

C Variables

Variables are used to store values. Example: 

int age = 20;
float price = 99.5;
char grade = 'A';

C Data Types

Common data types:

  • int - integer
  • float - decimal
  • double - bigger decimal
  • char - single character
int a = 10;
float b = 5.5;
char c = 'X';

C Operators

Operators are used to perform operations:

  • Arithmetic: + - * / %
  • Relational: == != > < >= <=
  • Logical: && || !

C If...Else

If condition is true, code executes. 

if(age >= 18) {
   printf("Eligible");
} else {
   printf("Not Eligible");
}

C Switch

Switch is used for multiple choices. 

switch(day){
 case 1: printf("Mon"); break;
 case 2: printf("Tue"); break;
 default: printf("Invalid");
}
