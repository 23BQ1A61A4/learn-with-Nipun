C Introduction

C is a powerful programming language developed by Dennis Ritchie. It is used in operating systems, embedded systems and software development. It gives low-level memory control and high performance. C is structured and procedural language. It is the foundation for C++, Java and many modern languages.

#include <stdio.h>
int main(){
    printf("Hello World");
    return 0;
}

C Variables

Variables store data values. In C, variables must be declared with a data type. Common types: int, float, char. They help store user input and calculation results. Each variable occupies memory space.

#include <stdio.h>
int main(){
    int age = 25;
    printf("%d", age);
    return 0;
}

C Loops

Loops execute code multiple times. C supports for, while, do-while loops. Used in arrays and repetitive tasks. Improves efficiency. Reduces duplication of code.

#include <stdio.h>
int main(){
    for(int i=1;i<=5;i++){
        printf("%d ", i);
    }
    return 0;
}
